ICONO Y LOGO — Donde va cada archivo
====================================

1) Logo en la aplicacion (login, menu, splash, Excel)
   Archivo: src/renderer-react/public/logo.png
   Formato: PNG (preferible con transparencia).
   Tras cambiarlo:
     node scripts/enhance-logo-text.mjs
     npm run build:icon
     npm run build:installer-assets
   (o npm run dev para ver al instante en la app)

2) Icono de Windows (.exe, acceso directo, barra de tareas)
   Fuente:  build/app-icon-src.png  (se copia desde public/logo.png)
   Salida:  build/icon.ico
   Comando: npm run build:icon

3) Instalador NSIS (.exe setup)
   Fuente:  build/installer-logo-src.png  (misma copia del logo)
   Salidas: build/installer-sidebar.bmp, build/installer-header.bmp, build/installer-icon.ico
   Comando: npm run build:installer-assets

4) Manual PDF
   docs/manual-usuario/img/logo.png  (se sincroniza con sync-logo-assets.mjs)

Flujo completo antes de entregar al cliente:
   node scripts/enhance-logo-text.mjs
   npm run build:icon
   npm run build:installer-assets
   npm run build
