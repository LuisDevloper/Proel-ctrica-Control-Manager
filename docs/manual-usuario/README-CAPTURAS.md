# Capturas para el manual (cliente)

Guarde cada imagen en esta carpeta **`img/`** con **exactamente** el nombre de archivo indicado. Resolución recomendada: **1280–1600 px de ancho**, formato **PNG**.

| Archivo | Qué capturar | Acción que ilustra |
|---------|----------------|-------------------|
| `00-logo.png` | Opcional: logo o pantalla de bienvenida | Identificación del producto |
| `01-login.png` | Pantalla de inicio de sesión | Campos usuario/contraseña y botón de entrar |
| `02-dashboard.png` | Vista Dashboard con datos | Resumen y tarjetas KPI |
| `03-sidebar.png` | Menú lateral visible (opciones) | Navegación entre módulos |
| `04-motores-lista.png` | Lista de motores con filtros | Buscar y filtrar motores |
| `05-motores-nuevo.png` | Formulario de alta de motor | Crear un motor nuevo |
| `06-motores-detalle.png` | Vista detalle de un motor | Ver historial del motor |
| `07-motores-importar.png` | Modal importar Excel / plantilla | Importar desde Excel |
| `08-motores-exportar.png` | Barra de filtros con exportar | Exportar a Excel o PDF |
| `09-mantenimientos.png` | Lista o formulario mantenimiento | Registrar mantenimiento |
| `10-fallas.png` | Lista o formulario falla | Registrar falla |
| `11-tecnicos.png` | Lista técnicos | Alta/edición técnico |
| `12-inventario.png` | Lista inventario | Stock y mínimos |
| `13-calendario.png` | Calendario mensual | Ver mantenimientos por fecha |
| `14-notificaciones.png` | Campana abierta con alertas | Consultar notificaciones |
| `15-configuracion.png` | Pantalla Configuración | Info sistema, backup, actualizaciones |
| `16-usuarios.png` | Módulo Usuarios (solo admin) | Gestionar usuarios |
| `17-actividad.png` | Registro de actividad (solo admin) | Auditoría |

**Consejos:** recorte ventanas sensibles (si hace falta); use el **tema** que quiera mostrar al cliente; mantenga la misma **versión** de la app que entregará.
